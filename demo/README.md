# Playground Management REST API

A Spring Boot REST API for managing playground play sites with attractions, children, and queue management.

## How to Run

### Prerequisites
- Java 21 or higher
- Maven (or use the included Maven wrapper)

### Running the Application

1. Build the project:
   ```bash
   mvnw.cmd clean install
   ```

2. Run the application:
   ```bash
   mvnw.cmd spring-boot:run
   ```

The API will be available at `http://localhost:8080`
